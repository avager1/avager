JAVA IO流

流的分类

按方向

输入流：将文件中的内容读入到程序中

输出流：将程序中的内容写入到文件中

按单位

字节流：以字节为单位，可以读写所有数据

字符流：以字符为单位，只能读写文本数据

按功能

字节流：具有实际传输数据的读写功能

过滤流：在节点流的基础之上增强功能

字节流

字节流的父类（抽象类）

InputStream：字节输入流

package com.zw.demo;

import java.io.FileInputStream;


/*
 * 演示FileInputStream的使用
 * 文件字节输入流
 */
public class text_01 {
	public static void main(String[] args) throws Exception {
		// 1、创建FileInputStream,并指定文件路径
		FileInputStream fis = new FileInputStream("F:\\aaa.txt");//得f盘有这个文件
		// 2读取文件
		// 2.1一个字节一个字节读取
//		int data = 0;
//		while ((data = fis.read()) != -1) {
//			System.out.println((char)data);
//		}
		//2.2一次读取多个字节
		byte[] buf=new byte[1024];
		int count=0;
		while((count=fis.read(buf))!=-1) {
			System.out.println(new String(buf,0,count));
		}
		
		
		// 3关闭
		fis.close();
		System.out.println("执行完毕");
	}

}

文件字节流

文件复制：

package com.zw.demo;
/**
 * 使用文件字节流实现文件的复制
 * 
 */
import java.io.FileInputStream;
import java.io.FileOutputStream;

public class text_03 {
public static void main(String[] args) throws Exception{
	//创建FileInputStream，并设置路径
	FileInputStream fis=new FileInputStream("f:\\001.jpg");
	//创建FileOutputStream，并设置路径
	FileOutputStream fos=new FileOutputStream("f:\\002.jpg");
	//2一次读取多个字节
	byte[] buf=new byte[1024*4];
	int count=0;
	//边读边写
	while((count=fis.read(buf))!=-1) {
		fos.write(buf,0,count);
	}
	//关闭
	fis.close();
	fos.close();
}
}

字节缓冲流

缓冲流：BufferedInputStream/BufferedOuputStream

提高IO效率，减少访问磁盘次数

数据存储在缓冲区，flush是将缓存区的内容写入文件中，也可以直接close

BufferedInputStream的使用:

package com.zw.demo;

import java.io.BufferedInputStream;
import java.io.FileInputStream;

/**
 * 使用字节缓冲流读取
 * 可以理解为加强FileInputStream
 * @author 14226
 *
 */
public class text_11 {
	public static void main(String[] args) throws Exception {
		// 1.1、创建FileInputStream,并指定文件路径
		FileInputStream fis = new FileInputStream("F:\\aaa.txt");
		// 1.2、创建BufferedInputStream
		BufferedInputStream bis = new BufferedInputStream(fis);
		// 2读取文件
//			int data = 0;
//			while ((data = bis.read()) != -1) {
//				System.out.println((char)data);
//			}

		byte[] buf = new byte[1024];
		int count = 0;
		while ((count = bis.read(buf)) != -1) {
			System.out.println(new String(buf, 0, count));
		}

		// 3关闭
		bis.close();
		System.out.println("执行完毕");
	}
}

BufferedOuputStrea的使用:

package com.zw.demo;

import java.io.BufferedOutputStream;
import java.io.FileOutputStream;

/**
 * 使用字节缓冲流写入
 * 
 * @author 14226
 *
 */
public class text_12 {
	public static void main(String[] args) throws Exception {
		// 1创建FileOutputStream，并设置路径
		FileOutputStream fos = new FileOutputStream("f:\\bbb.txt");
		BufferedOutputStream bos = new BufferedOutputStream(fos);
		// 2写入文件
		for (int i = 0; i < 10; i++) {
			bos.write("helloword\t".getBytes());// 写入8k的缓存区中（数据小于8k，不会直接写入bbb.txt中，需使用bos.flush()）
			bos.flush();// 刷新到硬盘
		}
		// 3关闭（内部调用flush方法）
		bos.close();
	}
}

对象流

ObjectOutputStream/ObjectIntputStream

增强缓冲区功能

增强读写8种基本数据类型和字符串功能

增强了读写对象的功能：

​ readObject() 从流中读取一个对象

​ writeObject(Object obj) 向流中写入一个对象

使用流传输对象的过程成为序列化、反序列化

使用ObjectOutputStream实现对象的序列化：

Student类：

package com.zw.demo;

import java.io.Serializable;

public class Student implements Serializable {

	/**
	 * Serializable：序列化版本号ID
	 */
	// private static final long serialVersionUID = 1L;
	private String name;
	private int age;

	// private transient int age;//使用transient修饰属性，这个属性不能被序列化

	@Override
	public String toString() {
		return "Student [name=" + name + ", age=" + age + "]";
	}

	public Student() {
		super();
	}

	public Student(String name, int age) {
		super();
		this.name = name;
		this.age = age;
	}

	public String getName() {
		return name;
	}

	public void setName(String name) {
		this.name = name;
	}

	public int getAge() {
		return age;
	}

	public void setAge(int age) {
		this.age = age;
	}

}

实现对student的序列化：

package com.zw.demo;

import java.io.FileOutputStream;
import java.io.ObjectOutputStream;
import java.util.ArrayList;

/**
 * 使用ObjectOutputStream实现对象的序列化 序列化：就是将对象转化成二进制字节码 要求：
 * 1、对序列化的类student上implements Serializable，Serializable是一个标记接口，代表可序列化
 * 2、序列化版本号ID，保证序列化的类和反序列化的类是同一个类 
 * 3、序列化类中对象属性要求实现Serializable接口
 * 4、使用transient（瞬时的）修饰属性，这个属性不能被序列化
 * 5、静态属性不能序列化
 * 6、序列化多个对象，可以借助集合实现
 * 
 * @author 14226
 *
 */

public class text_ObjectOutputStream {

	public static void main(String[] args) throws Exception {
		// 创建ObjectOutputStream对象流，是在FileOutputStream基础上
		FileOutputStream fos = new FileOutputStream("f:\\student.bin");
		ObjectOutputStream oos = new ObjectOutputStream(fos);
		// 序列化（写入）
		Student stu1 = new Student("张三", 23);
		Student stu2 = new Student("李四", 21);
		ArrayList<Student> list = new ArrayList<>();
		list.add(stu1);
		list.add(stu2);
		// oos.writeObject(stu1);
		oos.writeObject(list);
		// 关闭
		oos.close();
		System.out.println("序列化完毕");
	}
}

实现对student的反序列化：

package com.zw.demo;

import java.io.FileInputStream;
import java.io.ObjectInputStream;

/**
 * 使用ObjectInputStream实现反序列化（读取重构成对象）
 * 
 * @author 14226
 *
 */
public class text_ObjectInputStream {

	public static void main(String[] args) throws Exception {
		// 创建对象流
		FileInputStream fis = new FileInputStream("f:\\student.bin");
		ObjectInputStream ois = new ObjectInputStream(fis);
		// 读取文件（反序列化）
		Student stu2 = (Student) ois.readObject();
		// 关闭
		ois.close();
		System.out.println("执行完毕");
		System.out.println(stu2);

	}

字符编码

ISO-8859-1:收录除ASCII外，还包括西欧，希腊语，泰语，阿拉伯语，希伯来语对应的文字符号

UTF-8：针对Unicode码表的可变长度字符编码

GB2312：简体中文

GBK：简体中文、扩充

BIG5：台湾，简体中文

文件字符流

使用FileReader和FileWrite复制文本文件，不能复制图片或二进制文件

FileReader：

​ public int read(char[] c) //从流中读取多个字符，将读到的内容存入c数组，返回实际读到的字符数；如果达到文件的尾部，则返回-1。

案例：

package com.zw.demo;

import java.io.FileReader;

/**
 * 使用字符流FileReader读取文件 
 * 记得修改记事本和编译器的编码，该例子是UTF-8，否则会出现中文乱码
 * 
 * @author 14226
 *
 */
public class text_FileReader {
	public static void main(String[] args) throws Exception {
		// 1、创建字符流
		FileReader fr = new FileReader("f:\\hello.txt");

		// 2、读取文件
		// 2.1、单个字符读取
//		int data = 0;
//		while ((data = fr.read()) != -1) {//读取一个字符
//			System.out.print((char) data);
//		}
		// 2.2一次读取多个字节
		char[] buf = new char[3];
		int count = 0;
		while ((count = fr.read(buf)) != -1) {
			System.out.println(new String(buf, 0, count));
		}

		// 3、关闭
		fr.close();
	}
}

FileWriter：

​ public void write(String str) //一次写多个字符，将b数组中所有字符，写入输出流。

案列：

package com.zw.demo;

import java.io.FileWriter;

/**
 * 使用FileWriter写入文件
 * 
 * @author 14226
 *
 */
public class text_FileWriter {
	public static void main(String[] args) throws Exception {
		// 创建FileWriter对象
		FileWriter fw = new FileWriter("f:\\hello-write.txt");
		// 写入
		fw.write("与去看是非，不如看你");
		fw.flush();
		// 关闭
		fw.close();
		System.out.println("执行完毕");
	}
}
